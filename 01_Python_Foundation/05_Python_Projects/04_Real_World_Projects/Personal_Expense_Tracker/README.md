# Personal Expense Tracker

A command-line Personal Expense Tracker built using Python.

## Overview

This project helps users manage their daily expenses, track spending habits, and generate expense reports.

The application stores expense data in CSV files and provides features such as expense tracking, searching, monthly summaries, category-wise analysis, and report exporting.

## Features

* Add Expense
* View Expenses
* Search Expenses
* Monthly Summary
* Category-wise Spending
* Export Expense Report

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* CSV File Handling
* Exception Handling
* Modular Programming

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd Personal_Expense_Tracker
```

3. Run the application

```bash
python main.py
```
